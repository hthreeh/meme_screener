## 🔑 核心发现

### 1. 固定50%止盈已是最优

- 50%止盈触发率24.8%，平衡了收益和风险
- 更低的止盈线(30%)虽然触发率高，但利润薄
- 更高的止盈线(80%+)触发率太低

### 2. 移动止盈反而更差！

| 移动止盈策略 | vs 固定50%止盈 |
|-------------|---------------|
| 涨30%后回撤10% | ❌ -21.8% |
| 涨30%后回撤15% | ❌ -25.9% |
| 涨50%后回撤20% | ❌ -14.9% |

**原因分析**：
- 市场波动剧烈，等回撤触发时往往跌更多
- 固定止盈能更早锁定利润

### 3. 分段止盈效果一般

- 30%减仓50%，60%清仓：-4.2%
- 50%减仓50%，100%清仓：-2.9%
- 不如直接50%一次性止盈

## 💡 最终结论

### 最优组合策略

| 机制 | 最佳方案 | 效果 |
|------|---------|------|
| **止损** | 分段止损(-15%/-30%) | +126% 🏆 |
| **止盈** | 固定50%止盈 | 基准 |
| **超时** | 趋势延期(涨10%+) | +9.4% |

### 完整策略配置

```python
OPTIMAL_CONFIG = {
    # 分段止损（核心改进）
    'staged_stop_loss': {
        'level_1': (-0.15, 0.5),  # 跌15%减仓50%
        'level_2': (-0.30, 1.0),  # 跌30%清仓
    },
    
    # 固定止盈（保持不变）
    'take_profit': 0.50,  # 涨50%止盈
    
    # 趋势延期（可选）
    'trend_extension': {
        'threshold': 0.10,  # 涨10%触发
        'extension': 30,    # 延期30分钟
        'max_times': 2,     # 最多延期2次
    },
    
    # 基础超时
    'base_timeout': 30,  # 分钟
}
```
