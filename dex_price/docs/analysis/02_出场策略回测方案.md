## 📊 数据分析结论

### 1. 不同超时时间的收益分析

| 超时时间 | 平均收益 | 胜率 | 结论 |
|---------|---------|------|------|
| 30分钟 | -0.0055 SOL | 33.1% | 当前默认，表现较差 |
| **60分钟** | **-0.0020 SOL** | 32.3% | ⭐ **表现最佳** |
| 90分钟 | -0.0087 SOL | 27.7% | 开始恶化 |
| 120分钟 | -0.0127 SOL | 28.9% | 继续恶化 |
| 180分钟 | -0.0133 SOL | 26.9% | 明显变差 |
| 240分钟 | -0.0126 SOL | 25.3% | 无改善 |

> **重要**: 数据显示**固定延长超时时间并不能改善收益**，60 分钟已是最优。真正的问题在于**何时该延长、何时该退出**。

### 2. 上升趋势判断标准（数据驱动）

| 30分钟涨幅 | 60分钟平均变化 | 样本数 | 建议 |
|-----------|--------------|-------|------|
| 跌超5% | **-8.8%** | 72 | 🔴 立即止损 |
| 跌0-5% | +9.9% | 24 | 观望 |
| 涨0-5% | +2.3% | 21 | 观望 |
| 涨5-10% | +6.8% | 12 | 可延期 |
| **涨10-20%** | **+41.9%** | 17 | ✅ **强烈延期** |
| **涨20%+** | **+60.0%** | 37 | ✅ **强烈延期** |

**上升趋势定义**：
```
趋势判断规则：
- 当前市值 >= 买入市值 × 1.10（涨 10%+）→ 延期超时
- 当前市值 < 买入市值 × 0.95（跌 5%+）→ 不延期，触发止损判断
```

| 100k-200k | 53分钟 | 47分钟 | 50分钟 |
| **200k-500k** | **90分钟** | 72分钟 | **75分钟** |
| 500k-1M | 83分钟 | 52分钟 | 60分钟 |
| >1M | 78分钟 | 64分钟 | 65分钟 |

**市值分段规则**：
```python
def get_timeout_by_market_cap(market_cap):
    if market_cap < 100000:      # <100k
        return 45  # 分钟
    elif market_cap < 200000:    # 100k-200k
        return 50
    elif market_cap < 500000:    # 200k-500k
        return 75  # 最佳市值段
    elif market_cap < 1000000:   # 500k-1M
        return 60
    else:                        # >1M
        return 65
```
