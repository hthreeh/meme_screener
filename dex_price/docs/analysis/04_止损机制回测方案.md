| 回撤范围 | 数量 | 最终盈利 | 最终亏损 | 反弹率 |
|---------|------|---------|---------|-------|
| 0-10% | 52 | 42 | 10 | **80.8%** |
| 10-15% | 26 | 22 | 4 | **84.6%** |
| 15-20% | 33 | 14 | 19 | 42.4% |
| 20-25% | 35 | 24 | 11 | 68.6% |
| 25-30% | 41 | 18 | 23 | 43.9% |

> **关键发现**：
> - 跌幅 0-15% 时，反弹率高达 80-85%，**不应立即止损**

> - 15-30% 是关键区间，需要更智能的判断

## 🧪 回测方案设计

### 测试场景

| 场景 | 描述 | 预期效果 |
|------|------|---------|
| A. 基准 | 固定-30%止损 | 对照组 |
| B. 分段止损 | -15%减仓50%，-30%清仓 | 预期改善 |
| C. 回撤15%止损 | 从最高点回撤15%触发 | **预期最优** |
| D. 回撤20%止损 | 从最高点回撤20%触发 | 对比 |
| E. 组合策略 | 回撤15% + 趋势延期 | 综合测试 |

### 分段止损规则

```python
def staged_stop_loss(buy_mc, current_mc, position_ratio=1.0):
    """分段止损"""
    multiplier = current_mc / buy_mc
    
    if multiplier <= 0.85 and position_ratio > 0.5:  # 跌15%
        return 0.5  # 减仓50%
    elif multiplier <= 0.70:  # 跌30%
        return 0    # 清仓
    
    return position_ratio  # 保持仓位
```

### 回撤止损规则

```python
def trailing_stop_loss(buy_mc, current_mc, highest_mc, threshold=0.15):
    """从最高点回撤止损"""
    if highest_mc <= buy_mc:
        return False  # 未曾上涨，不触发
    
    drawdown = (highest_mc - current_mc) / highest_mc
    return drawdown >= threshold
```
