## 🏆 最优方案：分段止损

### 规则

```python
def staged_stop_loss(buy_mc, current_mc, position=1.0):
    multiplier = current_mc / buy_mc
    
    if multiplier <= 0.85 and position > 0.5:  # 跌15%
        return 0.5  # 减仓50%
    elif multiplier <= 0.70:  # 跌30%
        return 0    # 清仓
    
    return position
```

### 效果

| 指标 | 基准(-30%) | 分段止损 | 变化 |
|------|-----------|---------|------|
| 总 PnL | -2.4058 | **+0.6289** | **+3.03 SOL** |
| 平均 PnL | -0.0048 | +0.0013 | ⬆️ |
| 胜率 | 35.7% | 34.7% | ≈ |

> **从亏损变成盈利！** 改善幅度达 **126%**

## 📈 方案详细分析

### 为什么分段止损效果最好？

1. **跌15%时减仓50%**
   - 保护了一半本金
   - 如果继续下跌，损失减半
   - 如果反弹，仍有50%仓位获利

2. **跌30%才清仓**
   - 给予充分反弹空间
   - 避免过早止损错过反弹

3. **数据支撑**
   - 跌10-15%时反弹率高达 84.6%
   - 跌30%+后反弹率仅 10.6%

### 回撤止损分析

| 方案 | 效果 | 原因 |
|------|------|------|
| 回撤10%止损 | ❌ 更差 | 过于敏感，频繁误杀 |
| 回撤15%止损 | ✅ 略好 | 平衡点 |
| 回撤20%止损 | ❌ 更差 | 反应迟钝 |

**结论**：回撤止损效果不如分段止损

## 🔑 核心发现

### 1. 分段止损 >> 回撤止损

- 分段止损：+126.1%
- 回撤15%止损：+5.9%
- 差距巨大！

### 2. 固定-20%止损值得考虑

- 改善23.6%，规则简单
- 适合不想复杂逻辑的场景

### 3. 组合策略效果一般

- 回撤15%+趋势延期：+6.3%
- 不如单独分段止损

```python
# position_tracker.py 修改建议
def check_stop_loss(self, position, current_price):
    multiplier = current_price / position.buy_price
    
    if multiplier <= 0.85:  # 跌15%
        if position.remaining_ratio > 0.5:
            self.sell(position, ratio=0.5, reason='staged_stop_15')
    
    if multiplier <= 0.70:  # 跌30%
        self.sell(position, ratio=1.0, reason='stop_loss_30')
```
